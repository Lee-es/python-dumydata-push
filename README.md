# MTM Dummy 데이터 자동 전송 도구

헬스 키오스크 테스트를 위한 **더미 데이터 자동 전송 프로그램**입니다.\
Python + uv 기반으로 동작하며, 비개발자도 쉽게 실행할 수 있도록
제작되었습니다.

------------------------------------------------------------------------

## 1. 준비물

###  1) Python 설치 (필수)

1.  아래 링크에서 Python 3.10 이상 다운로드\
    👉 https://www.python.org/downloads/
2.  설치 시 다음 옵션 **꼭 체크**
    -   **"Add Python to PATH"**
3.  설치 후 확인

``` cmd
python --version
```

------------------------------------------------------------------------

### ✔️ 2) pip 업그레이드

``` cmd
python -m pip install --upgrade pip
```

------------------------------------------------------------------------

### ✔️ 3) uv 설치

``` cmd
pip install uv
```

설치 확인:

``` cmd
uv --version
```


------------------------------------------------------------------------

## 2. Windows 실행 방법

### **1) run_dummy_data.bat 찾기**

프로젝트 폴더 안에서 `run_dummy_data.bat` 확인.

### **2) 더블클릭 실행**

-   uv가 자동으로 가상환경 생성
-   필요한 패키지 설치
-   main.py 자동 실행
